# Flyway migrations

Versioned SQL migrations for the `dev` and `prod` profiles (real PostgreSQL). The `test`
profile runs the same migrations against H2 in PostgreSQL-compatibility mode, so keep SQL
ANSI-compatible where practical (e.g. `GENERATED BY DEFAULT AS IDENTITY` instead of
Postgres-only extensions).

No domain migrations yet — this foundation milestone has no entities. Add the first one as
`V1__<description>.sql` when domain work starts.
